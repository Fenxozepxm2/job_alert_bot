# Job Alert Bot 🚀🤖

Асинхронный Telegram-бот для умного поиска, агрегации и фильтрации вакансий (текущая интеграция — **hh.ru**). Проект создается как высокотехнологичная альтернатива существующим карьерным ботам с фокусом на глубокую кастомизацию и отсечение «мусорных» предложений.

> ⚠️ **Статус проекта:** В активной разработке (Active Development). Основное ядро и работа с БД уже запущены, идет расширение функционала.

---

## 🔥 Реализованный функционал
*   **Сохранение пользовательских фильтров:** Бот умеет запоминать сложные критерии поиска вакансий под конкретного пользователя.
*   **Асинхронный парсинг API:** Эффективное взаимодействие с API HeadHunter без блокировки основного потока бота.
*   **Персистентность данных:** Использование ORM для надежного хранения настроек пользователей, истории поиска и логов вакансий.
*   **Управление миграциями:** Интеграция полноценной системы миграций базы данных для гибкого изменения архитектуры приложения.

---

## 🛠️ Технологический стек
*   **Язык:** Python 3.11+
*   **Фреймворк бота:** Aiogram 3.x (Асинхронный подход)
*   **База данных и ORM:** SQLAlchemy, PostgreSQL / SQLite
*   **Миграции:** Alembic
*   **Интеграции:** HeadHunter API (hh.ru), TOML конфигурации

---

## 🗺️ Дорожная карта проекта (Roadmap)
Проект активно развивается. Ближайшие этапы реализации:

- [x] Создание архитектуры проекта и подключение БД через SQLAlchemy.
- [x] Настройка системы миграций Alembic.
- [x] Базовая интеграция с API hh.ru и сохранение фильтров.

## 💻 Локальное развертывание

### 1. Клонирование репозитория
```bash
git clone https://github.com
cd job_alert_bot
```

### 2. Настройка окружения
Создайте файл конфигурации на основе примера:
```bash
cp settings.example.toml settings.toml
```
Заполните `settings.toml` вашим токеном бота и параметрами подключения к БД.

### 3. Запуск миграций и бота
```bash
# Установка зависимостей
pip install -r requrement.txt

# Применение миграций базы данных
alembic upgrade head

# Запуск приложения
python main.py
```
